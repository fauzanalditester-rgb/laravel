# ERP System - Technical Documentation

## 🏗️ Architecture & Stack
- **Framework**: Laravel 11.x
- **Frontend**: Blade Templates + Tailwind CSS v4
- **Database**: MySQL 8.x
- **Authentication**: Laravel Breeze + customized Spatie Permissions
- **PDF Generation**: `barryvdh/laravel-dompdf`

## 🗄️ Database Structure (ERD Overview)

### 1. `users`
- Stores system users.
- Relasi: `hasMany` ActivityLogs, Penjualan, LaporanKas, dll (as `created_by`).
- Relasi: `hasRoles` (via Spatie).

### 2. `rekap_timbangans`
- **Main Operational Table**.
- Fields: `tanggal`, `nomor_kendaraan`, `jenis_material`, `berat_masuk`, `berat_keluar`, `berat_bersih`.
- Tracks raw material entry via trucks.

### 3. `penjualans`
- **Sales Transactions**.
- Fields: `nomor_invoice`, `customer`, `jenis_material`, `jumlah`, `total_harga`, `status_bayar`.
- Status: `paid`, `unpaid`, `partial`.

### 4. `laporan_kas`
- **Cash Ledger**.
- Fields: `tanggal`, `jenis` (in/out), `kategori`, `jumlah`, `saldo`.
- Using "Running Balance" calculation logic.

### 5. `rekap_leburs`
- **Production Result**.
- Fields: `berat_awal`, `berat_hasil`, `susut`, `persentase_susut`.
- **Logic**: If `persentase_susut` > 10%, triggers Anomaly Alert on Dashboard.

### 6. `activity_logs`
- **Audit Trail**.
- Stores every `created`, `updated`, `deleted` event from observable models.
- Fields: `user_id`, `action`, `module`, `old_values`, `new_values`, `ip_address`.

## 🚀 Deployment Guide (VPS / Server)

### Prerequisites
- PHP 8.2+
- Composer
- MySQL/MariaDB
- Nginx/Apache

### Steps
1. **Clone & Install**
   ```bash
   git clone <repo_url>
   cd project-folder
   composer install --optimize-autoloader --no-dev
   npm install && npm run build
   ```

2. **Environment Setup**
   ```bash
   cp .env.example .env
   # Edit .env database credentials
   php artisan key:generate
   ```

3. **Database Migration**
   ```bash
   php artisan migrate --seed
   # Seeder creates default Super Admin: abi@erp.com / password
   ```

4. **Permissions**
   ```bash
   chown -R www-data:www-data storage bootstrap/cache
   chmod -R 775 storage bootstrap/cache
   ```

5. **Optimization (Production Only)**
   ```bash
   php artisan config:cache
   php artisan route:cache
   php artisan view:cache
   ```

6. **Queue Worker (Optional but Recommended for Logs)**
   - Setup Supervisor to run `php artisan queue:work`.

### SSL & Security
- Use **Certbot** for free Let's Encrypt SSL.
- Ensure only port 80/443 are open.
- Set `APP_DEBUG=false` in production.
